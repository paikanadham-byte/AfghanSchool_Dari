مدرسه آنلاین افغانستان — نسخهٔ آزمایشی (دری)
===========================================
این یک نسخهٔ آزمایشی محلی است که به زبان دری طراحی شده است. 
شامل:
- مرور کتاب‌ها بر اساس صنف و موضوع
- مشاهده PDFهای عمومی (پیوندها)
- افزودن کتاب توسط مدیر (URL یا آپلود PDF)
- سوالات تستی چهار گزینه‌ای (هر سوال یک جواب درست)
- اسکریپر ساده برای جمع‌آوری PDFهای عمومی از دامنهٔ MOE

اجرای محلی (Local testing)
---------------------------
نیازمندی‌ها:
- Node.js (نسخهٔ 16+)
- npm

مراحل:
1. پوشه را باز کنید:
   cd /path/to/AfghanSchool_Dari
2. نصب وابستگی‌ها:
   npm install
3. اجرای crawler (اختیاری — جمع‌آوری فایل‌های PDF عمومی):
   npm run crawl
4. اجرای سرور:
   npm start
5. باز کردن مرورگر به آدرس:
   http://localhost:4000

یادآوری حقوقی:
- این نسخه فقط فایل‌هایی را که عمومی و روی سایت وزارت/منبع منتشر شده است فهرست می‌کند.
- هنگام بارگذاری فایل‌های محتوایی، اطمینان حاصل کنید حقوق نشر را دارید.
